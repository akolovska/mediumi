Основи на Python

Логичките услови во Python се користат за донесување на одлуки во кодот. Тие овозможуваат извршување на различни делови од кодот во зависност од тоа дали даден услов е ‘True’ или ‘False’.

Логичките услови се имплементираат со користење на условни изрази како: if, elif, else.


If - else циклуси

if израз – се користи за проверка дали дадениот услов е точен. Доколку е точен, кодот во тој блок се извршува

______________________________________________________

elif израз – elif претставува кратенка за “else if“, се користи за да се проверат дополнителни услови ако првиот if услов не е точен.
Пример


Објаснување
Ако променливата vreme_vo_denot е помала од 12, ќе се испечати "Добро утро!".
Ако vreme_vo_denot не е помала од 12, но е помала од 18, ќе се испечати "Добар ден!".
Ако ниеден од овие услови не е вистинит, значи времето е после 18:00 и ќе се испечати "Добра вечер!".

______________________________________________________

else израз – се користи за да се изврши дополнителен код, ако ниту еден од претходните улови не е точен.


Објаснување
Ако променливата temperatura е поголема од 30, ќе се испечати "Многу е топло надвор."
Ако temperatura не е поголема од 30, но е поголема од 20, ќе се испечати "Времето е пријатно."
Ако ниту еден од претходните услови не е точен, односно ако temperatura е 20 или помала, ќе се испечати "Малку е ладно надвор."

______________________________________________________

PASS

Исказот if не може да биде празен, но ако поради некоја причина имате потреба истиот да биде празен (без содржина), ставете го изразот PASS за да не добиете грешка!!
