Матрицата е збирка на броеви распоредени во правоаголна низа во редови и колони. Во областа на инженерството, физиката, статистиката и графиката, матриците се широко користени за изразување на ротации на слики и други видови трансформации. Матрицата се нарекува m со n матрица, означена со симболот „m x n“ ако има m редови и n колони. Матрица со 4 колони и 3 реда. Оваа матрица е 3x4 (се изговара „три по четири“) матрица бидејќи има 3 реда и 4 колони.
Пајтон нема вграден тип за матрици. Сепак, можеме да ја третираме листата на листа како матрица. На пример:
A = [[1, 4, 5],
[-5, 8, 9]]
Оваа листа на листа може да ја третираме како матрица која има 2 реда и 3 колони.

Собирање на две матрици

Ние користиме + оператор за да додадеме соодветни елементи на две NumPy матрици.
import numpy as np A = np.array([[2, 4], [5, -6]])
B = np.array([[9, -3], [3, 6]])
C = A + B # додавање на елементи
print(C)
'''
Output:
[[11 1]
[ 8 0]]
'''

Множење на матрици


За да множиме две матрици, користиме метода dot().
Забелешка: * се користи за множење низи (множење на соодветните елементи од две низи) не за множење на матрици.
import numpy as np
A = np.array([[3, 6, 7], [5, -3, 0]])
B = np.array([[1, 1], [2, 1], [3, -3]])
C = A.dot(B)
print(C)
'''
Output:
[[ 36 -12]
[ -1 2]]
'''

Пристап до елементи во матрица


Слично како списоците, можеме да пристапиме до елементите на матрицата користејќи index(). Да почнеме со еднодимензионална NumPy низа.
import numpy as np
A = np.array([[1, 4, 5, 12],
[-5, 8, 9, 0],
[-6, 7, 11, 19]])
# Првиот елемент од првиот ред
print("A[0][0] =", A[0][0])
# Третиот елемент од вториот ред
print("A[1][2] =", A[1][2])
# Последниот елемент од последниот ред
print("A[-1][-1] =", A[-1][-1])
A[0][0] = 1
A[1][2] = 9
A[-1][-1] = 19

Пристап до редици

import numpy as np
A = np.array([[1, 4, 5, 12],
[-5, 8, 9, 0],
[-6, 7, 11, 19]])
print("A[0] =", A[0]) # Прва редица
print("A[2] =", A[2]) # Трета редица
print("A[-1] =", A[-1]) # Последна редица

A[0] = [1, 4, 5, 12]
A[2] = [-6, 7, 11, 19]
A[-1] = [-6, 7, 11, 19]

Пристап до колони

import numpy as np
A = np.array([[1, 4, 5, 12],
[-5, 8, 9, 0],
[-6, 7, 11, 19]])
print("A[:,0] =",A[:,0]) # Прва колона
print("A[:,3] =", A[:,3]) # Четврта колона
print("A[:,-1] =", A[:,-1]) # Последна колона

A[:,0] = [ 1 -5 -6]
A[:,3] = [12 0 19]
A[:,-1] = [12 0 19]

Опсег во матрици


Опсег во матрици се користи исто како и во низи, со методот slice().
import numpy as np
A = np.array([[1, 4, 5, 12, 14],
[-5, 8, 9, 0, 17],
[-6, 7, 11, 19, 21]])
print(A[:2, :4]) # две редици, четири колони
''' Output:
[[ 1 4 5 12]
[-5 8 9 0]]
'''
print(A[:1,]) # прва редица, сите колони
''' Output:
[[ 1 4 5 12 14]]
'''
print(A[:,2]) # сите редици, втора колона
''' Output:
[ 5 9 11]
'''
print(A[:, 2:5]) # сите редици, трета до петта колона
'''Output:
[[ 5 12 14]
[ 9 0 17]
[11 19 21]]
'''
