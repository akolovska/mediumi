Init конструктор


Претходно му доделивме стандардна вредност на атрибутот класа,
class Bike:
name = ""
...
# create object
bike1 = Bike()

Сепак, можеме да иницијализираме вредности користејќи ги конструкторите. На пример,
class Bike:
# constructor function
def __init__(self, name = ""):
self.name = name
bike1 = Bike()

Овде, __init__() е конструкторската функција која се повикува секогаш кога ќе се инстанцира нов објект од таа класа.
Конструкторот погоре ја иницијализира вредноста на атрибутот name.
Го користевме self.name за да се однесуваме на атрибутот име на објектот bike1.
Ако користиме конструктор за иницијализирање на вредности во класата, треба да ја пренесеме соодветната вредност при креирањето на објектот на класата.
bike1 = Bike("Mountain Bike")

Овде, „Mountain Bike“ се пренесува на параметарот за името на __init__().
