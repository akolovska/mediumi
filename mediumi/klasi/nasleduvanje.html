Наследување во Python

Наследувањето ни овозможува да дефинираме класа која ги наследува сите методи и својства од друга класа. Матичната класа е класата од која се наследува, исто така наречена основна класа. Детска класа е класата што наследува од друга класа, наречена и изведена класа.
Секоја класа може да биде матична класа, така што синтаксата е иста како и креирањето на која било друга класа:
Креирајте класа со име Person, со својства за име и презиме и метод за печатење:
class Person: def __init__(self, fname, lname): self.firstname = fname self.lastname = lname def printname(self): print(self.firstname, self.lastname) #Use the Person class to create an object, and then execute the printname method: x = Person("John", "Doe") x.printname()
За да креирате класа што ја наследува функционалноста од друга класа, испратете ја матичната класа како параметар кога ја креирате класата дете:
Направете класа со име Student, која ќе ги наследи својствата и методите од класата Person:
class Student(Person): pass
Забелешка: Користете го клучниот збор pass кога не сакате да додавате други својства или методи во класата.
Сега класата Student ги има истите својства и методи како класата Person.
Користете ја класата Student за да креирате објект, а потоа извршете го методот printname:
x = Student("Mike", "Olsen") x.printname()
Досега создадовме дете класа која ги наследува својствата и методите од својот родител.
Сакаме да ја додадеме функцијата __init__() во класата дете (наместо клучниот збор pass).
Забелешка: Функцијата __init__() се повикува автоматски секогаш кога класата се користи за создавање нов објект.
class Student(Person): def __init__(self, fname, lname): #додади атрибути
Кога ќе ја додадете функцијата __init__(), класата дете повеќе нема да ја наследува функцијата __init__() на родителот.
Забелешка: Функцијата __init__() на детето го надминува наследувањето на функцијата __init__() на родителот.
За да го задржите наследството на функцијата __init__() на родителот, додајте повик до функцијата __init__() на родителот:
class Student(Person): def __init__(self, fname, lname): Person.__init__(self, fname, lname)
Сега успешно ја додадовме функцијата __init__() и го задржавме наследството на матичната класа и подготвени сме да додадеме функционалност во функцијата __init__().
Пајтон, исто така, има функција super() која ќе ја натера дете класата да ги наследи сите методи и својства од својот родител:
class Student(Person): def __init__(self, fname, lname): super().__init__(fname, lname)
Со користење на функцијата super(), не мора да го користите името на матичниот елемент, тој автоматски ќе ги наследи методите и својствата од неговиот родител.
Додајте својство наречено graduationyear во класата Student:
class Student(Person): def __init__(self, fname, lname, year): super().__init__(fname, lname) self.graduationyear = 2019
Во примерот подолу, 2019 година треба да биде променлива и да се префрли во класата Student при креирање студентски објекти. За да го направите тоа, додадете друг параметар во функцијата __init__():
Додадете параметар year:
class Student(Person): def __init__(self, fname, lname): super().__init__(fname, lname) self.graduationyear = year x = Student("Mike", "Olsen", 2019)
Додадете метод наречен welcome во класата Student:
class Student(Person): def __init__(self, fname, lname, year): super().__init__(fname, lname) self.graduationyear = year def welcome(self): print("Welcome", self.firstname, self.lastname, "to the class of", self.graduationyear)
Ако додадете метод во класата дете со истото име како функција во матичната класа, наследувањето на родителскиот метод ќе се отфрли.
