Објекти на Python


Објектот се нарекува инстанца на класа.
Да претпоставиме дека Bike е класа, тогаш можеме да креираме објекти како bike1, bike2, итн од класата.
Еве ја синтаксата за креирање објект.
objectName = ClassName()

Ајде да видиме пример,
# создај класа
class Bike:
name = ""
gear = 0
# креирај објект од класата
bike1 = Bike()
Овде, bike1 е објектот на класата. Сега, можеме да го користиме овој објект за пристап до атрибутите на класата.

Пристап до атрибутите на класата користејќи објекти


Ние ја користиме . нотација за пристап до атрибутите на класата. На пример,
# изменете го името
bike1.name = "Mountain bike"
# пристап до опремата
bike1.gear
Овде, ги користевме bike1.name и bike1.gear за промена и пристап до вредноста на атрибутите на име и опрема, соодветно.
# define a class
class Bike:
name = ""
gear = 0
# create object of class
bike1 = Bike()
# access attributes and assign new values
bike1.gear = 11
bike1.name = "Mountain Bike"
print(f"Name: {bike1.name}, Gears: {bike1.gear} ")

Во горниот пример, ја дефиниравме класата со име Bike со два атрибути: име и опрема.
Создадовме и објект bike1 од класата Bike.
Конечно, пристапивме и ги изменивме својствата на објектот користејќи го . нотација.

Креирање повеќе инстанци од класа


# класа
class Employee:
employee_id = 0
# две инстанци од класа
employee1 = Employee()
employee2 = Employee()
# пристап до атрибутот од првата инстанца
employee1.employeeID = 1001
print(f"Employee ID: {employee1.employeeID}")
# пристап до атрибутот од втората инстанца
employee2.employeeID = 1002
print(f"Employee ID: {employee2.employeeID}")
Output:
Employee ID: 1001
Employee ID: 1002
Можеме да дефинираме и функција во класата на Python. Функцијата Python дефинирана во класата се нарекува метод.
# create a class
class Room:
length = 0.0
breadth = 0.0
# method to calculate area
def calculate_area(self):
print("Area of Room =", self.length * self.breadth)
# create object of Room class
study_room = Room()
# assign values to all the properties
study_room.length = 42.5
study_room.breadth = 30.8
# access method inside class
study_room.calculate_area()
Run Code
Output
Area of Room = 1309.0
Во горниот пример, создадовме класа со име Room со:
Атрибути: должина и ширина
Метод: пресметај_површина()
Овде, создадовме објект со име study_room од класата Room.
Потоа го користевме објектот за доделување вредности на атрибути: должина и ширина.
Забележете дека исто така го користевме објектот за да го повикаме методот во класата,
study_room.calculate_area()

Еве, ние го користевме . нотација за да го повикате методот. Конечно, изјавата во методот се извршува.
