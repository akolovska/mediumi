Класата се смета за план на објекти. Можеме да го замислиме часот како скица (прототип) на куќа. Ги содржи сите детали за подовите, вратите, прозорците итн. Врз основа на овие описи, ја градиме куќата; куќата е објектот. Бидејќи многу куќи можат да се направат од истиот опис, можеме да создадеме многу објекти од класа.

Дефиниција на класа во Python


Го користиме клучниот збор class за да креираме класа во Python. На пример,
class className:
...
Овде, создадовме класа со име ClassName.
Ајде да видиме пример,
class Bike:
name = ""
gear = 0
Овде,
Bike - името на класата
name/gear - променливи внатре во класата со стандардни вредности „“ и 0 соодветно.
Забелешка: Променливите во класата се нарекуваат атрибути.
