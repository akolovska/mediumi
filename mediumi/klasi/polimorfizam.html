Полиморфизам

Полиморфизмот често се користи во методите на класи, каде што можеме да имаме повеќе класи со исто име на методот. На пример, да речеме дека имаме три класи: Car, Boat и Plane, и сите тие имаат метод наречен move(): Различни класи со ист метод:
class Car:
def __init__(self, brand, model):
self.brand = brand
self.model = model
def move(self):
print("Drive!")
class Boat:
def __init__(self, brand, model):
self.brand = brand
self.model = model
def move(self):
print("Sail!")
class Plane:
def __init__(self, brand, model):
self.brand = brand
self.model = model
def move(self):
print("Fly!")
car1 = Car("Ford", "Mustang") #Креирајте класа Car
boat1 = Boat("Ibiza", "Touring 20") #Креирајте класа Boat
plane1 = Plane("Boeing", "747") #Креирајте класа Plane
for x in (car1, boat1, plane1):
x.move()
Погледнете ја јамката за на крајот. Поради полиморфизмот можеме да го извршиме истиот метод за сите три класи.

Полиморфизам на класа на наследување

Што е со часовите со детски паралелки со исто име? Можеме ли да користиме полиморфизам таму?
Да. Ако го искористиме примерот погоре и направиме родителска класа наречена Vehicle и направиме Car, Boat, Plane детски класи на Vehicle, детските класи ги наследуваат методите Vehicle, но можат да ги отфрлат:
Создадете класа наречена Vehicle и направете дете класи на возила за Car, Boat, Airplane:
class Vehicle:
def __init__(self, brand, model):
self.brand = brand
self.model = model
def move(self):
print("Move!")
class Car(Vehicle):
pass
class Boat(Vehicle):
def move(self):
print("Sail!")
class Plane(Vehicle):
def move(self):
print("Fly!")
car1 = Car("Ford", "Mustang") #Create a Car object
boat1 = Boat("Ibiza", "Touring 20") #Create a Boat object
plane1 = Plane("Boeing", "747") #Create a Plane object
for x in (car1, boat1, plane1):
print(x.brand)
print(x.model)
x.move()
Детските класи ги наследуваат својствата и методите од матичната класа.
Во примерот погоре можете да видите дека класата Car е празна, но го наследува brand, model и move() од Vehicle.
Класите Boat и Plane исто така наследуваат brand, model и move() од Vehicle, но и двете го отфрлаат методот move().
Поради полиморфизмот можеме да го извршиме истиот метод за сите класи.
