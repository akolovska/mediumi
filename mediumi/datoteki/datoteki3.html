За да напишете во постоечка датотека, мора да додадете параметар во функцијата open():
"a" - Додади - ќе се додаде на крајот на датотеката
„w“ - Write - ќе ја презапише секоја постоечка содржина
Отворете ја датотеката „demofile2.txt“ и додајте ја содржината на датотеката:

f = open("demofile2.txt", "a")
f.write("Now the file has more content!")
f.close()
#open and read the file after the appending:
f = open("demofile2.txt", "r")
print(f.read())
Отворете ја датотеката „demofile3.txt“ и презапишете ја содржината:
f = open("demofile3.txt", "w")
f.write("Whoops! I have deleted the content!")
f.close()
#open and read the file after the overwriting:
f = open("demofile3.txt", "r")
print(f.read())
За да креирате нова датотека во Python, користете го методот open(), со еден од следниве параметри:
"x" - Креирај - ќе создаде датотека, враќа грешка ако датотеката постои
"a" - Додади - ќе создаде датотека ако наведената датотека не постои
"w" - Write - ќе создаде датотека ако наведената датотека не постои
Креирајте празна датотека со име "myfile.txt":
f = open("myfile.txt", "x")
Креирајте нова датотека која не постои:
f = open("myfile.txt", "w")
