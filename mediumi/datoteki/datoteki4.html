Избришете датотека

За да избришете датотека, мора да го увезете модулот за ОС и да ја извршите неговата функција os.remove():
Отстранете ја датотеката „demofile.txt“:
import os
os.remove ("demofile.txt")
За да избегнете да добиете грешка, можеби ќе сакате да проверите дали датотеката постои пред да се обидете да ја избришете:
Проверете дали постои датотека, а потоа избришете ја:
import os
if os.path.exists ("demofile.txt"):
os.remove("demofile.txt")
else:
print(„Датотеката не постои“)

Избриши папка

За да избришете цела папка, користете го методот os.rmdir():
Отстранете ја папката „myfolder“:
import os
os.rmdir („myfolder“)
