Методи за работа кај множествата

add() -> Додава елемент во множеството

clear() -> Ги брише сите елементи од множеството

copy() -> Враќа копија од множеството

difference() -> Го враќа множество кое ги содржи разликите помеѓу две или повеќе множества (-)

difference_update() -> Ги брише елементите од тековниот сет, што ги има во друго,специфицирано множество (-=)

discard -> Го брише селектираниот елемент

insertsection() -> Го враќа елементот, според наведениот индекс (&)

insertsection_update() -> Го брише елементите, што ги нема во другп,специфицирано множество (&=)

isdisjoint() -> Враќа дали две множества имаат пресек или не

issubset() -> Враќа дали друго множество го содржи ова множество или не (<=)

issuperset() -> Враќа дали ова множество, содржи друго множество или не (>=)

pop() -> Брише елемент од множеството

remove() -> Брише некој специфициран елемент

symmetric_difference() -> враќа множество со симетрични разлики од две множества (^)

symmetric_difference_update() -> Ги вметнува симетричните разлики од ова во друго множество (^=)

union() -> Враќа множество кое содржи унија на множества (|)

update() -> Го ажурира множеството со унијата на ова множество и други (|=)
